CSS

  • CSS stands for cascading Style Sheets
  • It describes how the HTML elements are to be displayed on the screen or in other media.
  • It also saves a lot of works and it can control the layout of multiple web pages at once.
  • Language used to specify the presentation aspects of structurally marked up documents.
  • Developed by Halkon Wium Lie (CHSS) and Bert Bos (SSP).
