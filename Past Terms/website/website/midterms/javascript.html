Javascript

  • Javascript is a lightweight scripting language used in web development, it provides multiple functionalities that allows user interraction and behavior modification.
  • JavaScript can modify and change:
      • HTML Content
      HTML Attributes
      HTML Styles
      HTML Elements
  • A JavaScript can be inserted in an HTML file using a script tag placed either inside a head tag or body tag, it can also linked in an external javascript resource indicated with a (.js) file format in the head tags of the HTML file.
  • Scripts are executed in a top to bottom level approach
  • Functions are executed when invoked inside or outside the script

Advantages in Using Javascript

  • Decreased Server Load
      • User input can be validated without going over the server, which can save the server unecessary traffic
  • Quick feedback
      • Minimal page reloads.
  • Enhanced Interfaces
      • Javascript is able to give the website functionality and increased interactivity.
  • Inexpensive to Use
      • Javascript can be coded via a text editor

Limited Features of JavaScript

  • Client-side Javascipt cannot read or write files
  • No support available for networking applications
  • No multithreading or multiprocessor capabilities

Language & API

  • Traditional Browser Object Model (BOM) API
  • Document Object Model (DOM) API

DOM (Document Object Model)

  • The DOM model is an abstract model in the form of a logical tree containing multiple branches ending in a node.
  • The nodes in the DOM model contain objects that define the interface between the HTML documents and application programs
  • Essential DOM Methods include:
    • getElementById Method
      • Accesses elements via id elements
    • innerHTML
      • Property that can modify and access the contents of an HTML elements
    • Node methods
      • node.childNodes
      • node.firstChild
      • node.nextSibling
      • node.nextSibling
      • node.parentNode
    • Event Handler Methods

BOM (Browser Object Model)

  • The BOM allows communication between the script and the browser and uses the window object and window properties to modify the state of the websites contents inside the browser window
  • Unlike DOM, BOM doesen't currently have any standardization.

Javascript Syntax

  • Spaces, indentations and linebreaks will be ignored
  • Semicolon optional
  • Loose Datatyping
  • Case Sensitive
  • Uses a C-style or C++-style comments
  • Keywords should not be used to name a variable

Datatypes
